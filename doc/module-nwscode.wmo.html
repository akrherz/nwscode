C WmoHeader(...) ...

A World Meteorological Organization Abbreviated Header parser.

For a full description of the WMO Header format see WMO Manual's 306 and 386. The National Weather Service has made much of this information available at the following address:

http://www.weather.gov/tg/headef.html

Attributes:

designator
Designator for the contents of the bulletin.
station
The identification of the processing center that generated the bulletin.
issuance
The product issuance time.
addendum
The optional BBB group.
